# Remocon for pooq

## 소개

- 키보드로 Pooq를 편리하게 이용할 수 있습니다. 
- 키 입력하여 전체화면, 채널 변경, 재생목록 보기 지원

## 다운로드 
- [Remocon for Pooq @Chrome Web Store](https://chrome.google.com/webstore/detail/remocon-for-pooq/fanlfkdjgojjhjigakefkncagnfiekel)
